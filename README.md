# 🚆 Rail Hazard AI (YOLOv8 Project)

## Description
Real-time object detection system using YOLOv8 and OpenCV.  
Simulates railway monitoring and hazard detection using video streams.

## Features
- Real-time object detection
- Webcam/video processing
- Bounding box visualization

## Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- PyTorch

## Run Instructions
```bash
pip install -r requirements.txt
python main.py
